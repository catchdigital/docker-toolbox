# docker-toolbox

Basic image to use for builds.

This image is based on php 8.1-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.9.23
* aws cdk: 2.64.0
* aws amplify: 10.7.2
* eb: 3.20.3
* php: 8.1.15-fpm
* node: v18.14.0
* npm: 9.4.2
* composer: 2.5.3
* python: 2.7.18
* python3: 3.9.2
* ansible: 2.14.2
