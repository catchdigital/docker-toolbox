# docker-toolbox

Basic image to use for builds.

This image is based on php 8.2-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.17.20
* aws cdk: 2.150.0
* aws amplify: 12.12.4
* eb: 3.20.10
* php: 8.2.21-fpm
* node: v20.16.0
* npm: 10.8.1
* composer: 2.7.7
* python3: 3.11.2
* ansible: 2.17.2
