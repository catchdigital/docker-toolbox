# docker-toolbox

Basic image to use for builds.

This image is based on php 8.3-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.22.7
* aws cdk: 2.171.1
* aws amplify: 12.13.1
* eb: 3.21.0
* php: 8.3.14-fpm
* node: v20.18.1
* npm: 10.8.2
* composer: 2.8.3
* python3: 3.11.2
* ansible: 2.18.0
* acli: 2.35.1
