# docker-toolbox

Basic image to use for builds.

This image is based on php 8.2-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.13.35
* aws cdk: 2.108.0
* aws amplify: 12.8.2
* eb: 3.20.10
* php: 8.2.12-fpm
* node: v20.9.0
* npm: 10.1.0
* composer: 2.6.5
* python3: 3.11.2
* ansible: 2.16.0
