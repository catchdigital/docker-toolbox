# docker-toolbox

Basic image to use for builds.

This image is based on php 8.0-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.4.28
* aws cdk: 2.17.0
* aws amplify: 7.6.26
* eb: 3.20.3
* php: 8.0.12-fpm
* node: v14.19.1
* npm: 8.5.5
* composer: 2.2.9
* python: 3.9.2
* ansible: 2.12.3
