# docker-awscli

Basic image to use for builds.

This image is based on php 8.0-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.3.2
* aws cdk: 1.129.0
* aws amplify: 6.3.1
* eb: 3.20.2
* php: 8.0.12-fpm
* node: v14.18.1
* npm: 8.1.2
* composer: 2.1.9
* python: 3.9.2
